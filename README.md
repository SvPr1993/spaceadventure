<h1 align="center">Spaceadventure Project</h1>

Описание проекта для GitHub

Сайт некой вымышленной космической компании, которая занимается организацией космических экспедиций

## Используемые технологии

- **Django**: Веб-фреймворк для разработки проекта.

# Основные функции
 - возможность внесения данных в специальную форму для отправки в базу данных (капсула времени).
 - возможность выбора шрифта для внесенных данных.
